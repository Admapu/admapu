Bienvenido a la documentación de Admapu. Este proyecto busca desarrollar una prueba de concepto (PoC) funcional de un token ERC-20 con características específicas de control y verificación.

## On-chain
- Token ERC-20 deployado
- Mint controlado por roles
- Transferencias restringidas a wallets verificadas
- Verifier contract (mock o real)
- Al menos un programa social on-chain

## Off-chain
- Flujo reproducible de deploy
- Scripts documentados
- Tests automáticos

## Demo
- Wallet no verificada NO puede transferir
- Wallet verificada SÍ puede transferir
