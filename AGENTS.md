# AGENTS.md - Project Guidelines

## Repository expectations
- Always run tests with `forge test -vv` before submitting any PR
- Document all the public utilities and deployments in `docs/` when chaging behavior. All the documentation must be written in Spanish

## Technical constraints
- Ensure 100% test coverage

## Branch naming convention
- Use Conventional Branch v1.0.0
- Always branch off `main` branch when starting a new feature or fix
